# Student Grade Calculator (Python)
 
*Overview*

A simple command-line Python program that computes the total marks, percentage and grade for a 
student from marks entered for multiple subjects.

 
*Features*

~ Accepts number of subjects and marks per subject. 
~ Easy command-line interface for entering marks.
~ Calculate total and percentage.
~ Assigns a grade according to percentage.
~ Grades : S,A,B,C,D,F

 
*Requirements* 

~ Python 3.8+ (works on Python 3.x) 
~ No third party packages  required.

 
*Usage*
 
1.Run the script.
2.Enter the number of subjects.
3.Enter the marks for each subject.
4.View the total, percentage, and grade.


*How it works*

~ Promts the user to input the number of subjects.
~ Collects marks for each subject.
~ Calculates total marks and percentage.
~ Determines grade based on percentage.
~ Prints results in a readable format.

python grade_calculator.py
