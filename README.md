# Student Grade Calculator (Python)
 
*Overview* 
A simple command- line Python program that computes the total marks, percentage and grade for a 
student from marks entered for multiple subjects.

 
*Features* 
~ Accepts number of subjects and marks per subject. 
~ Validates inputs (non-negative numbers, reasonable maximum). 
~ Computes total marks, percentage and assigns a grade:
     - S : >=  90 
     - A : >= 80 
     - B : >= 70 
     - C : >= 60 
     - D : >= 50 
     - F :  <  50   
~ Prints a formatted result summery.

 
*Requirements* 
~ Python 3.8+ (works on Python 3.x) 
~ No third party packages  required.

 
*Usage* 
1. Clone the repository or download the files. 
2. Run the script: 
```bash 
python grade_calculator.py
